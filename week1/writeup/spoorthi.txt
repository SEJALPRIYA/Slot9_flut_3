INTRODUCTION TO APP DEVELOPMENT WITH FLUTTER
WEEK 1
Softwares installed :
•	Flutter : Flutter is an open-source UI software development kit created by Google. It is used to develop applications for Android, iOS, Windows, Mac, Linux. Flutter uses Dart as the programming language.
•	Android Studio : It is an IDE for Android application development.
•	Git bash : Git Bash is an application for Microsoft Windows environments which provides an emulation layer for a Git command line experience.
Apps : A software application designed to run on a mobile device.
Components of an app:
1. Activities : The presentation layer of apps.
2. Services : They run at backend, updating the data sources.
3. Content providers : Used to manage and persist the application data.
4. Intents : inter application message passing framework
5. Broadcast Receivers : Enable app to listen to the intents that satisfy the matching criteria specified by us.
6. Widgets : Small visual app components that one can find on the home screen of the devices.
7. Notifications : App alerts used to draw user’s attention to some particular  app event without interrupting the current activity of the user.

App structure :
Scaffold widget implements the basic material design visual layout structure. Inside this, we have appbar, body of an app and the navigation bar.
